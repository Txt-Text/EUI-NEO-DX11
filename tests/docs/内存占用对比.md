# 内存占用对比

这份文档对应的是上游历史环境下的旧采样记录，不适用于当前这个 Windows + DX11 fork。

它原先基于非当前定位的构建路径和图形后端，继续保留具体数值只会制造误导，因此这里不再展示旧环境下的历史采样内容。


## 当前状态

- 本文档已停止作为有效性能基线使用
- 当前 fork 的内存数据需要在 Windows 环境下重新采样
- 采样对象应围绕当前 Win32 + DX11 渲染路径建立


## 后续建议

重新建立基线时，至少记录：

- Windows 版本
- CPU / GPU 型号
- `Debug` 或 `Release`
- `Win32` 或 `x64`
- 空窗口、基础渲染、真实页面动画三类场景
- RSS、Private Working Set、Commit、显存占用等口径
