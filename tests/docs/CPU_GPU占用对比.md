# CPU / GPU 占用对比

这份文档对应的是旧环境下的历史采样记录，不适用于当前这个 Windows + DX11 fork。

原文件中的测试口径建立在非当前定位的窗口后端、渲染后端和系统环境上，继续保留具体数值会误导阅读者，因此这里不再展示旧环境下的历史数据。


## 当前状态

- 本文档已停止作为有效性能基线使用
- 当前 fork 的 CPU / GPU 数据需要在 Windows 环境下重新采样
- 新基线应围绕当前 Win32 + DX11 渲染路径建立


## 后续建议

重新采样时，建议至少记录：

- Windows 版本
- CPU / GPU 型号
- `Debug` 或 `Release`
- `Win32` 或 `x64`
- 空窗口、基础渲染、真实页面动画三类场景
- CPU 占用、present 延迟、GPU 引擎占用、显存占用等口径
